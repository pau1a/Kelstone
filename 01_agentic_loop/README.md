# Agentic loop

How a model reasons, takes an action, observes the result, and repeats
— the basic shape underneath every "AI agent."

## Investigations

- **`deciding_when_to_use_a_tool/`** — a minimal agent (one tool, one
  instruction) examined at the exact moment it decides whether to call
  its tool or answer directly. Write-up: *"Inside the Moment an AI
  Decides"*.
