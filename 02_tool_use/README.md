# Tool use

How a model is given structured tools to call, and how it decides
between them — built from first principles, not a framework's
tool-calling API.

## Investigations

- **`two_real_tools_and_general_vs_specific_uncertainty/`** — two real
  tools (live weather and time APIs), a genuine two-stage decision, and
  an instrument separating a model's general linguistic uncertainty
  from its specific uncertainty about which tool to call.
